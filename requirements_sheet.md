# 🚀 Avdhi - Compliance Management System

### 🏗️ CORE MODULES
1. **User Management (RBAC)**: Admin, CA, and Client views.
2. **Smart Compliance Engine**: GST (11th/20th), TDS (7th), PF/ESIC (15th).
3. **Dashboard MIS**: "Smart Pipeline" tracking & "Audit Log" monitoring.
4. **CA Assignment**: Admin controls CA-Client relationships.

### 🛠️ TECH STACK
* **Frontend**: React + Vite + Framer Motion (Glassmorphism).
* **Backend**: FastAPI + SQLAlchemy + SQLite.
* **Logic**: Automated due-date calculation & alert seeding.

### 📦 FRONTEND DEPENDENCIES
* `lucide-react`
* `framer-motion`
* `react-router-dom`

### 📦 BACKEND DEPENDENCIES (Install these via pip)
* `fastapi`
* `uvicorn`
* `sqlalchemy`
* `pydantic[email]`
* `email-validator`
